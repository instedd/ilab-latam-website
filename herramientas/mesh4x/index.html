Mesh4X

Mesh4X es un paquete de servicios, aplicaciones y librerías que permiten que los datos se sincronicen offline a través de bases de datos, herramientas y dispositivos varios.

El desafío: Las soluciones tecnológicas aplicadas a atender necesidades sociales en ámbitos de muy bajos recursos o en situaciones de crisis extremas se enfrentan con el gran escollo que representan las conexiones a redes precarias e impredecibles, situación que complica significativamente el intercambio confiable de datos.


La herramienta: Mesh4X es un paquete de servicios, aplicaciones y librerías que permiten que la data que manipulas se sincronice offline a través de bases de datos, herramientas y dispositivos varios. Utiliza estándares para representar tus datos, y concilia las diferencias cada vez que dos fuentes de datos están interconectadas. Esto te permite visualizar y editar registros de datos a través de teléfonos celulares (y no meramente “reportar” datos), y combinar distintas bases de datos y mapas que han sido editados al mismo tiempo mientras los usuarios están fuera de línea. Mesh4X cuenta con adaptadores para archivos de Excel, bases de datos de Access, hojas de cálculo de Google, la mayoría de las bases de datos compatibles con Java NHibernate, archivos KML, EpiInfo y J2ME RMS.
